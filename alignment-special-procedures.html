Vehicle-Specific Alignment Procedures

Honda S2000  |  Acura MDX Type-S 2022+

Honda

S2000 — Front Alignment

Caster can be adjusted independently of camber — set caster first.

1

 Pre-alignment inspection — ride height, tire pressure, suspension wear

2

 Mount alignment heads — record initial readings

3

 ★ Adjust CASTER first — front eccentric bolt, lower control arm; rotate to spec; torque

4

 Verify caster adjustment did not affect camber reading

5

 Adjust camber — rear eccentric bolt, lower control arm; torque

6

 Set front toe — adjust tie rods equally; center steering wheel

7

 Final verification — print report; reset steering angle sensor if required

Acura

MDX Type-S 2022+ — Air Suspension Setup

Set BASE height & Leveling Off mode before any alignment work begins.

1

 ★ Set suspension to BASE height — engine running; use vehicle height selector; wait for system to fully settle

2

 ★ Enable Leveling Off mode — HDS — Adaptive Damper System menu; disables auto height correction during alignment

3

 Measure ride height all 4 corners — confirm BASE spec from iN; document on RO

4

 Pre-alignment inspection — suspension components, tire pressure, bearing play

5

 Mount alignment heads — record initial readings; perform standard 4-wheel alignment

6

 Print alignment report — confirm all angles within OEM spec

7

 ★ Disable Leveling Off mode — HDS — restore full air suspension operation; confirm normal system activity
