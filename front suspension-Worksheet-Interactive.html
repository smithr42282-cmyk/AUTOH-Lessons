Section 3 — Slide 3: Honda Inspection Procedure

Q8
Slide 3 — Honda Application

The lesson lists a 7-step inspection protocol in a specific order. The steps below are scrambled. Type the correct step number (1–7) in each box.

  • Ball joint check — use a pry bar under the tire; Honda specifies 0 mm radial play.
  • Ride height measurement — measure and compare to Honda specification.
  • Road test — note any pull, wander, noise location, or steering return concerns.
  • Control arm bushing check — inspect for cracking, tearing, or offset of the rubber.
  • Strut bounce test — vehicle should return and stop within 1–1.5 oscillations.
  • Visual inspection on lift — check for oil leaks, cracked boots, and corrosion.
  • Sway bar link check — grasp the link and check for looseness or end-play.
